## Overview

A personal **portfolio website** built to showcase my projects, skills, and experience as a **Full-Stack Web Developer**.  
The website highlights real projects, technical skills, and provides an easy way to get in touch.

---

## Features

 - Fully responsive design (mobile, tablet, desktop)

 - Modern hero section with quick access to CV & social links

 - About Me section with highlights and current role

 - Skills section (Frontend, Backend, Databases, Tools & Soft Skills)

 - Experience timeline (Orange & Coding Academy)

 - Projects section with:

   - Category tabs (All / Full Stack / Frontend / Backend)

- Interactive carousel navigation

 - Certifications & education section

 - Professional contact section with:

   - Email, phone, WhatsApp, LinkedIn

   - Message form (Email & WhatsApp integration)
